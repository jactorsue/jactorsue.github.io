设置ssh免密登陆

问题描述

有两台服务器A和B，如何让A通过ssh连接B的时候不用每次都输入密码。

方法

在A机器上用 ssh-keygen 生成私钥和公钥，将生成的公钥复制到B上就可以了。

在A上生成密钥

之后就可以免密登陆B机器了。
